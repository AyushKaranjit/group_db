database entities

entity description
degree list of degrees with a unique title and details, included a level eg MSc, BSc, BEng, or BA
examiner external examiner details with an ID, name, institution, contact details
marker lecturer details
student student details; number, name, age, contact info
module list of modules and their details with an active status Y/N default Y
degree_module list of modules on a degree, with a start date and semester either S1 or S2
module_list list of students on a module
attendance list of students' attendance on a module, on a set date, default today
assignment assignment outline for a module, with a code, weighting and type
submission module assignment submission, with receipt no, date and time
student_submission a list of student's individual submissions, with their grade
